# Poly Chat Frontend

Static HTML, CSS, and TypeScript workbench UI for comparing local language
models. React can be introduced later if the UI grows enough to justify it.

## Run locally

Install Deno, then run:

```bash
cd frontend
deno task dev
```

The app expects the backend API at `http://localhost:8000`.

## Useful commands

```bash
deno task check
deno task build
```
